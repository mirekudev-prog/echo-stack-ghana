💡 How the bypass works

Your admin_session cookie is recognised by the server on every protected page (/app, /dashboard, /creator, /post/*). You are never redirected to the user login or signup screen while this cookie is active. Clicking "Home (set preview cookie)" also sets an extra admin_preview cookie for the public homepage.
